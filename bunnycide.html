Bunnycide

A fun 2d physics platform puzzle game for iOS
Developing Bunnycide
Bunnycide was the name of a 2D puzzle game that I created for fun as part of a two man team. I was the developer and my friend created the original design, assets and sounds. Your goal in this game was to save the bunny from certain death by manipulating objects in the scene. The game used a 2d physics engine to simulate the world and objects in it. Each level contained a new puzzle for the player to solve and each chapter a new kind of peril for the bunny.
We created roughly 30 levels for this game. I found it very challenging and time consuming to design, create and test levels that were fun and novel.
